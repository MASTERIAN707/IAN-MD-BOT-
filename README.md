# IAN-MD-BOT 🚀

Welcome to **IAN-MD-BOT**, created by MASTER.  
This WhatsApp bot is modular, powerful, and ready to deploy on **Railway** or **Render**.

## Deploy Instantly
[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new/template?repo=https://github.com/MASTER/IAN-MD-BOT)  
[![Deploy on Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/MASTER/IAN-MD-BOT)

## Features
- Modular command system (240+ commands planned)
- Shining auto-rewriting messages ✨
- Session pairing with WhatsApp (QR code)
- Easy fork + deploy setup

## Setup
1. Fork this repo
2. Add your `.env` file with WhatsApp API credentials
3. Deploy on Railway or Render using the buttons above
